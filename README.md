# French Kiss English Animated Text Page

This version keeps the zeppelin and uses the original composition.

Static French text areas are covered by matching overlay panels and replaced with animated English HTML text.

## Included

- Original artwork with zeppelin
- English navigation
- English zeppelin text
- English welcome banner
- Live countdown on the main board
- Rotating English menu text
- Rotating English side sign text
- Animated bottom slogan
- Interactive central board click and Colleague Space button

## Change the tarte time

Open `script.js` and edit:

```js
tarteHour: 15,
tarteMinute: 0,
```
